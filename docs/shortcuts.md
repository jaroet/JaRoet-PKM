# Keyboard Shortcuts

JaRoet PKM is designed to be used primarily with the keyboard.

### Global Navigation
| Shortcut | Action |
| :--- | :--- |
| **Arrow Keys** | Move focus between notes and sections. |
| **Shift + ← / →** | Switch between tabs in a focused pane (e.g., Siblings/Mentions). |
| **Alt + ← / →** | Navigate Back and Forward in your viewing history. |
| **Enter** | Recenter the focus on the currently **Active Center Note**. |
| **Space** | Navigate *into* the currently focused note (makes it the new active note). |
| **/** | Focus the global Search bar. |
| **Esc** | Close modals, exit search, or clear the current selection. |
| **Ctrl + J** | Jump to today's Journal note. |
| **Ctrl + Alt + R** | Go to a Random Note. |

### Selection & Actions
| Shortcut | Action |
| :--- | :--- |
| **x** | Toggle selection for the focused note and automatically move to the next one. |
| **Shift + Up/Down** | Select a range of notes in a list. |
| **F2** | Rename the focused note. |
| **Ctrl + Backspace** | **Delete** the focused note(s) permanently (with confirmation). |

### Linking & Organizing
*The Linker functions work relative to the **Currently Focused Note**. If no note is specifically focused (halo effect), they apply to the Central Note.*

| Shortcut | Action |
| :--- | :--- |
| **Ctrl + ↑** | Link to a **Parent**. (Opens Linker Modal). |
| **Ctrl + ↓** | Link to a **Child**. (Opens Linker Modal). |
| **Ctrl + ←** | Link to a **Related** note. (Opens Linker Modal). |
| **Backspace** | **Unlink** the focused note from the central note. |

### Editor
| Shortcut | Action |
| :--- | :--- |
| **Shift + Enter** | Open Editor in **Preview Mode**. |
| **Ctrl + Enter** | Open Editor in **Edit Mode**. |

**Inside the Editor:**
| Shortcut | Action |
| :--- | :--- |
| **Shift + Enter** | If in Edit Mode: **Save** and switch to Preview. If in Preview Mode: **Close** the editor. |
| **Ctrl + Enter** | If in Preview Mode: Switch to **Edit Mode**. If in Edit Mode: **Cancel/Close** without saving. |
| **[[** | Trigger the **Internal Link Autocomplete** popup. |
